A BASE DO FRONT-END



Introdução ao front-end

Estudando para ser front-end, você deverá conhecer três coisas: HTML, CSS e JavaScript. Conhecendo bem essas, você terá a base para seu currículo, mas se você deseja saber mais que a base vá no canal do Fiasco, do Curso em vídeo e Lucas Montano. Lá você conhecerá bem o que precisa saber, não só para ser o front-end, mas também para o back-end e ser um profissional cada vez mais qualificado podendo, se quiser, ser um fullstack.

HTML

HTML, que significa "Hypertext Markup Language" (Linguagem de Marcação de Hipertexto), é a linguagem padrão usada para criar e projetar páginas da web. Desenvolvido pelo World Wide Web Consortium (W3C), HTML permite a criação de documentos estruturados que incluem texto, links, imagens e outros elementos multimídia.

CSS

CSS, que significa "Cascading Style Sheets" (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para descrever a apresentação de um documento escrito em HTML ou XML. O CSS descreve como os elementos HTML devem ser exibidos na tela, no papel ou em outras mídias.

JavaScript

JavaScript é uma linguagem de programação de alto nível, interpretada e orientada a objetos. Originalmente, foi desenvolvida para ser executada no navegador web, permitindo a criação de páginas interativas. Com o tempo, o JavaScript expandiu seu alcance e agora é utilizado em diversos ambientes, incluindo servidores (Node.js) e aplicativos móveis.
